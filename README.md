# Victors Skispil

Et 2D skispil som webapp, hvor Victor skal lande det trick, banen kraever, for at komme videre.

## Indeholder

- 20 baner med stigende svaerhedsgrad
- Touch-kontrol til iPhone og iPad
- Pointsystem og progression
- Statisk opsaetning som kan deployes direkte til Vercel

## Lokal start

Aabn `index.html` direkte i en browser eller dobbeltklik paa `start-spillet.bat`.

## Deploy

Projektet er statisk og kan deployes direkte paa Vercel uden build-step.
